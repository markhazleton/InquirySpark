# DevSpark PowerShell Overrides

This folder is reserved for intentional team-level PowerShell script overrides.

Current state: no PowerShell overrides are active in this repository.

DevSpark resolves PowerShell scripts in this order:

1. `.documentation/scripts/powershell/`
2. `.devspark/scripts/powershell/`

Because this folder contains only this README, the repository currently uses the stock DevSpark v1.5 PowerShell scripts from `.devspark/scripts/powershell/`.
